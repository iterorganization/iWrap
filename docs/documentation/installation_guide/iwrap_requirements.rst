#######################################################################################################################
iWrap dependencies
#######################################################################################################################

Build and installation requirements
################################################


* Python interpreter: version 3.8 or newer (version preferred: 3.11)
* Python packages:

  * `setuptools`: version 43.0 or newer
  * `wheel`
  * `tomli`
  * `Sphinx`
  * `JupyterLab`
  * jupyter-book
  * jupyterlab_myst
  * myst-nb
  * myst-parser


Runtime dependencies
################################################

* Python interpreter: version 3.8 or newer (version preferred: 3.11)
* Python packages:

  * `pyyaml`
  * `lxml`
  * `jinja2`
  * `packaging`
  * `f90nml`
  * `jsonschema`
  * `jPype1`

* System libraries:

  * `JSON-Fortran`
  * `JsonCpp`
  * `Java` (1.8+)

* IMAS:

  * `IMAS Data Dictionary` (or `IMAS AL`)

Integration tests dependencies
################################################

* All runtime dependencies listed above
* Python packages:

  * `pylint`
  * `Dask`

* System libraries:

  * `XMLLib`
