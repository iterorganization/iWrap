iWrap requirements
#######################################################################################################################

**To build and install iWrap, the following requirements must be met:**

* *IMAS:*
.. code-block:: bash

   module load IMAS

* *LXML:*
.. code-block:: bash

   module load lxml

* *XMLLIB:*
.. code-block:: bash

   module load XMLlib

* *tomli:*
.. code-block:: bash

   python3 -m pip install --upgrade --user tomli
